    Function build

    Build and process a scaffold graph and collect the pile ups.

    ReadAlignment[][] build (
      in const(ulong) numReferenceContigs,
      AlignmentChain[] candidates,
      GapSegment[] inputGaps,
      in const(OptionsFor!(11)) options
    );

    Parameters

    NameDescription
    numReferenceContigs number of reference contigs used to construct the base graph which has four nodes for each contig. Contigs are assumed to have IDs 1 .. numReferenceContigs + 1.
    candidates alignment chains that are deemed reliable
    inputGaps list of intra-scaffold gaps that connect contigs of the input assembly
    options additional parameters (see CLI options for further information):
    • <in:reads>
    • <in:reference>
    • --best-pile-up-margin
    • --existing-gap-bonus
    • --debug-pile-ups
    • --no-merge-extension
    • --min-spanning-reads
    • --proper-alignment-allowance
    • --tmpdir
