    Module dentist.dazzler

    Defines bindings to and utilities for the Dazzler tool suite.

    Functions

    NameDescription
    alignmentChainPacker(localAlignments, bufferMode, localAlignmentBuffer) Transforms a range R of FlatLocalAlignments into a range of AlignmentChains.
    buildDamFile(fastaRecords, tmpdir, dbsplitOptions, append) Build outputDb with the given set of FASTA records. If no outputDb is given a temporary .dam file in tmpdir will be created.
    buildDbFile(fastaRecords, tmpdir, dbsplitOptions, append) Build outputDb with the given set of FASTA records. If no outputDb is given a temporary .db file in tmpdir will be created.
    chainLocalAlignments(dbFile, lasFile, options) Chain local alignments in lasFile and write back to a LAS file.
    computeQVs(dbFile, lasFile, coverage) Compute intrinsic quality values (QVs) from the alignments in lasFile.
    dazzExtra(name, data, accumMode) Represents a track extra, i.e. additional data stored alongside a DB track.
    dbdust(dbFile, dbdustOptions) Run DBdust on dbFile.
    dbEmpty(dbFile) Returns true iff dbFile is empty.
    dbSubset(inDbFile, readIds, options, append) Build DB file by using the given subset of reads in inDbFile. Writes to outputDB if given; otherwise a name with the same extenion as inDbFile under options.tmpdir is safely generated. The resulting DB is implictly split using options.dbsplitOptions.
    filterLocalAlignments(lasFile) Filter local alignments in lasFile producing a new LAS file.
    filterPileUpAlignments(dbFile, lasFile, properAlignmentAllowance, forceFlat) Disables alignments that should not appear in a pile up.
    getAlignments(dbA, lasFile, includeTracePoints) Read all alignment chains from lasFile.
    getBlockSize(dbFile) Return the of block size of dbFile.
    getConsensus(dbFile, readId, options) Build consensus using daccord.
    getContigCutoff(dbFile) Return the contig/read length cutoff of dbFile.
    getDalignment(dbFile, dalignerOptions, outdir) Align DB(s) to each other using daligner executed in outdir.
    getDamapping(refDb, queryDb, damapperOptions, outdir) Align queryDb against refDb using damapper executed in outdir.
    getDbRecords(dbFile, dbdumpOptions) Get the designated records of dbFile. Returns all records unless recordNumbers is given.
    getExactAlignment(dbA, dbB, ac, memoryLimit) [Experimental] Reconstruct the alignment matrix for ac force-filling gaps in the chain. Restrict reconstruction to [beginA, endA) on contig A if given.
    getFastaEntries(dbFile, recordNumbers, fastaLineWidth) Get the designated set of records in FASTA format. If recordNumbers is empty the whole DB will be converted.
    getFastaSequence(dbFile, recordNumber, cacheSize) Get the base pair sequence of the designated record.
    getFastaSequences(dbFile) Get the base pair sequences of the designated records. This does NOT include the header.
    getFlatLocalAlignments(lasFile, bufferMode, tracePointBuffer) Lazily read individual local alignments from lasFile.
    getHiddenDbFiles(dbFile) Return a list of hidden files associated to every .dam/.db file. These files contain the actual data used in all the computation. Thus, we carefully check for their existence.
    getLasFile(dbA, baseDirectory) Generate the LAS file name as daligner/damapper run in baseDirectory does.
    getLocalAlignments(dbA, options) Compute local alignments of given DBs using daligner. Uses dbB = dbA if dbB is omitted.
    getMappings(dbA, dbB, options) Compute local alignments of given DBs using damapper.
    getMaskFiles(dbFile, maskName, allowBlock) Get the hidden files comprising the designated mask.
    getNumBlocks(dbFile) Return the number of blocks in dbFile.
    getNumContigs(dbFile) Get the number of reads/contig in dbFile.
    getScaffoldStructure(dbFile) Returns a lazy range of ScaffoldSegments describing the scaffold structure.
    getTracePointDistance(lasFile) Returns the trace point distance in lasFile.
    isValidPileUpAlignment(alignment, allowance) An alignment in a pile up is valid iff it is proper and the begin/end of both reads match.
    LAmerge(mergedLas, lasFiles) Merge given las files.
    lasEmpty(lasFile) Returns true iff lasFile contains zero parts.
    lasFileGenerated(dbA, baseDirectory) Check if a LAS file was generated by daligner/damapper run in baseDirectory.
    numDbRecords(dbFile) Returns the (trimmed) number of records in dbFile.
    readDazzExtra(dbFile, maskName, extraName) Read extra extraName of track maskName with elements of type T from dbFile.
    readMask(dbFile, maskName) Read the Intervals of a Dazzler mask for dbFile.
    removeDB(dbFile) Remove database and hidden files.
    stripDbExtension(dbFile) Strip extension of dbFile if its is dbFileExtension or damFileExtension. Otherwise return dbFile untouched.
    withOption(dazzlerOptions, optionName, mod) Lazily modifies dazzlerOptions as specified.
    writeAlignments(lasFile, flatLocalAlignments) Write alignments to lasFile. This method takes care of the differences between alignment flags in DENTIST and Dazzler code.
    writeDazzExtra(dbFile, maskName, extra) Write extra extraName of track maskName with elements of type T from dbFile.
    writeMask(dbFile, maskName, intervals) Write the list of intervals to a Dazzler mask for dbFile.
    fingerprint(alignmentChain) Returns a tuple of contig IDs and first and last begin/end coords.
