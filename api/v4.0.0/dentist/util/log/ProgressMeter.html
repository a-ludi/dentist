    Struct ProgressMeter

    Tracks progress and outputs information regularly.

    struct ProgressMeter ;

    ProgressMeter.Format can be used to choose a format suitable for terminals (ProgressMeter.Format.human) or for log files (ProgressMeter.Format.json).

    Fields

    NameTypeDescription
    format ProgressMeter.FormatChoose display format.
    numTicks ulongNumber of ticks until now.
    precision ulongUse precision digits after the decimal point.
    printEveryMsecs ulongWait at least the amount of milliseconds before updating the status.
    silent std.typecons.Flag!("silent")Suppress output but still track the progress.
    totalTicks ulongSpecifies the 100% mark if given. No percentage is displayed if this is zero.
    unit ProgressMeter.UnitDisplay ticks in this unit.

    Properties

    NameTypeDescription
    elapsed[get] autoGet the number of elapsed timeUnits.
    eta[get] autoCalculate the estimated time of arrival (ETA).
    hasETA[get] boolTrue if the estimated time of arrival (ETA) can be calculated.
    output[set] std.stdio.FileSet the output file to write status updates to. Default it to use std.stdio.stderr.
    output[get] std.stdio.FileGet a reference to the output file.
    ticksPer[get] autoGet the average throughput in ticks per timeUnit.

    Methods

    NameDescription
    selectUnitFor (number) Select the smallest unit such that the number of decimal digits is up to three.
    start () Start the timer.
    stop () Stop the timer and print a last status update.
    tick () Add a single tick.
