    Module dentist.util.graphalgo

    A selection of graph algorithms.

    Functions

    NameDescription
    connectedComponents(n) Calculate connected components of the graph defined by hasEdge.
    dagSingleSourceShortestPaths(start, n) Calculate all shortest paths in a DAG starting at start.
    shortestPathsFloydWarshall(n) Calculate all shortest paths between all pairs of nodes. The functions hasEdge and weight define the graphs structure and weights, respectively. Nodes are represented as size_t integers.
    topologicalSort(n) Topologically sort a DAG.
