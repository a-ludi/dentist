    Function Region.this

    Construct a region from intervals.

    this (
      Region.TaggedInterval[] intervals
    );

    The newly constructed Region assumes ownership over intervals. Modifying them outside of the struct leads to undefined behavior.

    Example

    alias R = Region!(int, int);
alias TI = R.TaggedInterval;

auto region = R([TI(0, 15, 20), TI(0, 5, 10), TI(0, 0, 10)]);

// Intervals get implicitly normalized.
assert(region.intervals == [TI(0, 0, 10), TI(0, 15, 20)]);
