    Struct CompressedSequence

    A string of bases encoded in CompressedBaseQuads. The sequence may have any length – not just multiples of four.

    struct CompressedSequence ;

    Properties

    NameTypeDescription
    baseOffset[get] ubyteInternal property that is exposed for binary storage.
    bases[get] autoReturn a lazy range of the stored bases.
    compressedLength[get, set] ulongNumber of bytes required to store the sequence. This does not include meta data.
    data[get] const(CompressedBaseQuad[])Access underlying buffer directly. This is intended for binary storage.
    length[get] ulongReturns the number of bases in sequence.
