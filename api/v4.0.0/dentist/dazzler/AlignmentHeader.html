    Struct AlignmentHeader

    Meta information about a set of alignments, e.g. a LAS file.

    struct AlignmentHeader ;

    Usually this is created by reading a LAS file as efficiently as possible to improve performance of subsequent operations on the LAS file, e.g. by pre-allocating memory.

    Fields

    NameTypeDescription
    maxLocalAlignments ulongMaximum number of local alignments per chain
    maxLocalAlignmentsPerContig ulongMaximum total number of local alignments per contig
    maxTracePoints ulongMaximum number of trace points per local alignment
    numAlignmentChains ulongNumber of alignment chains (i.e. the unchained flag is unset)
    numAlignments ulongSum of number of alignment chains and number of unchained alignments
    numLocalAlignments ulongTotal number of local alignments disregarding chaining
    numTracePoints ulongTotal number of trace points
    tracePointDistance ulongTrace point distance
