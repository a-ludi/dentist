    Enum OptionModifier

    Specifies the type of action withOption should take.

    enum OptionModifier : ubyte { ... }

    Enum members

    NameDescription
    add Append new option.
    defaultValue Make sure option is present, e.i. an option with the given name exists. Does not modify existing options. This does nothing if no option value is given.
    ensurePresent Make sure option is present, e.i. option name and value must match.
    remove Remove existing options.
    replace Replace existing options. Does not add new option.
    replaceOrAdd Replace existing option or add as new option.
