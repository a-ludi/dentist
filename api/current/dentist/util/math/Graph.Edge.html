    Struct Graph.Edge

    An edge in the graph.

    struct Edge ;

    Edges may be directed or undirected, weighted or unweighted and have an additional payload or not.

    Constructors

    NameDescription
    this (start, end) Construct an edge.

    Fields

    NameTypeDescription
    _end Node
    _start Node
    payload EdgePayloadPayload associated with this edge. This is NOT taken into account in comparisons between edges.
    weight WeightWeight associated with this edge. This is taken into account in comparisons between edges.

    Properties

    NameTypeDescription
    end[get] NodeGet the end of this edge. For undirected graphs this is the larger of both incident nodes.
    start[get] NodeGet the start of this edge. For undirected graphs this is the smaller of both incident nodes.

    Methods

    NameDescription
    compareNodes (other)
    getConnectingNode (other) Returns the node that connects this edge with other edge. In case of undirected graphs this is just the common node of both edges; in directed case this is the end node of this edge if it matches the start node of other edge.
    opCmp (other) Orders edge lexicographically by start, end(, weight).
    opEquals (other) Two edges are equal iff their incident nodes (and weight) are the same.
    source (from) Get source of this edge beginning at node from. For undirected graphs returns the other node of this edge.
    target (from) Get target of this edge beginning at node from. For undirected graphs returns the other node of this edge.
