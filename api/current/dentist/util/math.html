    absdiff(a, b) Returns the absolute difference between two numbers.
    add(graph, edge) Add edge to graph and handle existing edges with handleConflict.
    bulkAdd(graph, edges) Add a set of edges to this graph and merge mutli-edges using merge.
    ceil(x, base) Round x towards +inf according to base, i.e. returns the next integer larger or equal to x which is divisible by base.
    ceildiv(a, b) Returns the result of ceil(a / b) but uses integer arithmetic only.
    filterEdges(graph) Filter edges of graph by pred in-place.
    findAllCliques(adjacencyList) Find all maximal cliques in a graph represented by adjacencyList. The implementation is based on version 1 of the Bron-Kerbosch algorithm [1].
    findCyclicSubgraphs(graph, incidentEdgesCache) Find a cycle base of an undirected graph using the Paton's algorithm.
    findMaximallyConnectedComponents(numNodes) Find all maximally connected components of a graph. The predicate isConnected will be evaluated O(n^^2) times in the worst-case and Ω(n) in the best case. In expectation it will be evaluated θ(n*log(n)).
    floor(x, base) Round x towards -inf according to base, i.e. returns the next integer smaller or equal to x which is divisible by base.
    longestIncreasingSubsequence(sequence) Calculate a longest increasing subsequence of sequence. This subsequence is not necessarily contiguous, or unique. Given a sequence of n elements the algorithm uses O(n log n) evaluation of pred.
    mapEdges(graph) Modify edges of graph with fun in-place.
    mean(values) Calculate the mean of values.
    mean(values, weights) Calculate the weighted mean of values.
    median(values) Calculate the median of values.
    N(values, totalSize) Calculate the Nxx (e.g. N50) of values.
