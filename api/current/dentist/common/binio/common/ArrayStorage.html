    Struct ArrayStorage

    An array-like structure of pointers that is used to access the data in a binary file. In this context, pointers are expected to be offsets in the binary file.

    struct ArrayStorage(T) ;

    Fields

    NameTypeDescription
    length size_tNumber of T items in this array.
    ptr size_tBase pointer to the first array element.

    Methods

    NameDescription
    fromPtrs (fromPtr, toPtr) Construct an ArrayStorage from given pointers. The pointers must be properly aligned.
    indexOf (ptr) Computes the index corresponding to ptr relative to this array. The result may be out of bounds. ptr must have a valid alignment relative to this.ptr.
    opDollar () Returns length.
    opIndex () Return a shallow copy if this.
    opIndex (i) Return the pointer to element 0 <= i <= length. Contrary to regular arrays, the element at i == length can be accessed. This is useful the construct slices (see opIndex(size_t[2])).
    opIndex (slice) Return a slice of this ArrayStorage. An empty slice is not allowed.
