    Module dentist.common.binio.common

    Collection of common structure and functions for working with binary data.

    Functions

    NameDescription
    lockIfPossible(file) Place a lock on file if the underlying file system supports it. This ignores and logs errors encountered while trying to acquire the lock. File locking may disabled entirely by setting the environment variable SKIP_FILE_LOCKING=1.
    readRecord(dbFile) Read a single record of type T at the current offset in dbFile.
    readRecordAt(dbFile, ptr) Read a single record of type T at offset ptr in dbFile.
    readRecords(dbFile, records) Read an array of records at the current offset in dbFile.

    Classes

    NameDescription
    BinaryIOException Thrown if an error occurs during binary I/O operations.

    Structs

    NameDescription
    ArrayStorage An array-like structure of pointers that is used to access the data in a binary file. In this context, pointers are expected to be offsets in the binary file.
    CompressedBaseQuad Four CompressedBases packed into one byte. Provides array-like access to these four bases.
    CompressedSequence A string of bases encoded in CompressedBaseQuads. The sequence may have any length – not just multiples of four.
