    Struct ReadAlignment

    Alignment of a read against the reference. This is either one or two alignment chains which belong to the same read and one or two reference contig(s).

    struct ReadAlignment ;

    It uses a static array of SeededAlignments to store the alignments, thus avoiding memory allocation.

    Constructors

    NameDescription
    this (alignments) Construct from one or two SeededAlignments. Returns an empty ReadAlignment If alignments has an invalid length and writes a dentist.util.log.LogLevel.debug message.

    Properties

    NameTypeDescription
    isAntiParallel[get] boolReturns true iff the read alignment spans a gap and the flanking contigs have in different orientation (according to this alignment).
    isBackExtension[get] boolReturns true iff the read alignment is an back extension, ie. it is an extension and reaches over the back of the reference contig.
    isExtension[get] boolReturns true iff the read alignment is an extension, ie. it is a front or back extension.
    isFrontExtension[get] boolReturns true iff the read alignment is an front extension, ie. it is an extension and reaches over the front of the reference contig.
    isGap[get] boolReturns true iff the read alignment spans a gap, ie. two alignments of the same read on different reference contigs are involved.
    isInOrder[get] boolIf readAlignment is a gap return true iff the first alignment's contigA.id is lower than the second alignment's; otherwise returns true.
    isParallel[get] boolReturns true iff the read alignment spans a gap and the flanking contigs have in the same orientation (according to this alignment).
    isValid[get] boolReturns true iff the read alignment is valid, ie. it is either an extension or gap.
    length[get] ulongReturns the number of SeededAlignments.
    opDollar[get] ulongReturns the number of SeededAlignments.
    type[get] ReadAlignmentTypeGet the type of the read alignment.
