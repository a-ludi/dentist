    Module dentist.commands.processPileUps.cropper

    This package contains methods for cropping a pile up.

    Example

━━━━┯ = reference contigs with trace points (top row)
──┾━━ = aligned reads with removed (thin) and kept (thick) parts
◊     = cropping positions
