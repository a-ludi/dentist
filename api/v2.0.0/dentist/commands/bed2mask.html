    Module dentist.commands.bed2mask

    The bed2mask command creates a Dazzler mask from a BED file.

    Command Summary

    Convert a BED file to a Dazzler mask. Implements ata comments-- a
special feature for DENTIST.

    Functions

    NameDescription
    execute(options) Execute the bed2mask command with options.
    getContigsByScaffold(refDb) Returns an associative array containing an array of ContigSegments for each scaffold identified by its FASTA header (without leading >).
    getOverlappingContigs(contigsByScaffold, scaffoldName, begin, end) Find contigs that overlap with the interval [begin, end] on scaffold.
    parseDataComment(comment, filename, lineNumber, contigIds, readIds) Parse data from comment into contigIds and readIds, respectively. If multiple parts of the same type are given later parts overwrite previous parts.
    readBedFile(bedFile, bedFileName, contigsByScaffold, parseDataComments) Main procedure that converts BED entries into an array of AugmentedReferenceIntervals. The returned intervals have Dazzler coordinates and contain data from the comment column if parseDataComments.
    writeDazzlerMask(refDb, maskName, augmentedMask, hasDataComments) Create Dazzler mask of intervals in augmentedMask adding extra tracks if hasDataComments is true.
